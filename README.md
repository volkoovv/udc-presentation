## 👀 Смотреть

**Живая версия:** https://volkoovv.github.io/udc-presentation/

Или локально — клонировать репозиторий и открыть через любой статический сервер:

```bash
python3 -m http.server 8099
# затем открыть http://127.0.0.1:8099/
```

> Просто дважды кликнуть по `index.html` (протокол `file://`) не сработает — деке нужен HTTP-сервер (она подгружает рантайм и ресурсы через `fetch`).

## 🎛 Навигация

- **Десктоп:** стрелки ← → / Space / PgUp / PgDn, клик по миниатюрам в левой ленте, `Home`/`End` — в начало/конец, `R` — сброс на первый слайд.
- **Планшет:** тап по правой/левой половине экрана + лента миниатюр.
- **Телефон:** вертикальный скролл — все 27 слайдов идут лентой.

Кликабельные ссылки внутри слайдов:
- 📊 живая финмодель пилота (Google Sheets);
- 📱 прототип приложения владельца — [udc-owner-app](https://volkoovv.github.io/udc-owner-app/);
- 📄 описание проекта в Notion.

## 🧱 Как это устроено

Самодостаточная HTML-дека в формате Claude Design (`.dc.html`):

| Файл | Роль |
|---|---|
| `index.html` | шаблон деки (27 слайдов) + инлайновые стили/анимации |
| `support.js` | рантайм Claude Design (парсит `<x-dc>`, подтягивает React) |
| `deck-stage.js` | веб-компонент сцены: масштабирование 1920×1080, навигация, адаптив |
| `assets/` | 18 изображений (фото и рендеры), оптимизированы под веб |

React/ReactDOM подгружаются с `unpkg.com` — нужен интернет при первом открытии.

## 📐 Адаптивность

- **≥ 641px + мышь** — режим сцены: слайд 1920×1080 масштабируется под экран, слева лента миниатюр.
- **планшет (тач)** — сцена с тап-навигацией, без ленты.
- **≤ 640px** — лента: слайды перестраиваются в вертикальный поток, картинки и блоки адаптируются под портрет.

Все проверено на 390×844 (телефон), 768×1024 (планшет) и 1440×900 (десктоп): горизонтального скролла нет, все 18 изображений грузятся, анимации входа отрабатывают.
