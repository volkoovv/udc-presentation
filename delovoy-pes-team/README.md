# Деловой пёс — презентация для команды

Версия общей презентации **без финансовых показателей** — для найма, онбординга и работы с командой.

**Живая версия:** https://volkoovv.github.io/udc-presentation/delovoy-pes-team/

Полная версия (с экономикой, для инвесторов и партнёров): https://volkoovv.github.io/udc-presentation/delovoy-pes/

## Что убрано относительно полной деки

**Удалены слайды целиком:**
- «Зрелый месяц» — выручка, contribution margin, EBITDA, микс выручки;
- «Структура капитала» — потребность в капитале, CAPEX, буфер, окупаемость.

**Убраны показатели с оставшихся слайдов:**

| Слайд | Было | Стало |
|---|---|---|
| Проект в одном слайде | капитал, окупаемость, ссылка на финмодель | режим полного дня 07:30–20:00, штат 4 ТД |
| Кинология | 84 + 42 = 126 тыс ₽/мес, % выручки | операционная функция и продукт удержания |
| Сервисы вокруг ядра | доли выручки 42/16/15/11% | роли: ядро продукта, после доверия, часть стандарта |
| Пилот «Парк Столиц» | аренда 247,5 тыс ₽/мес | режим клуба 07:30–20:00 |
| Dog-аккаунты | порог безубыточности 7,3/день | 3,7 визита на аккаунт в месяц |
| Go / No-Go | CAC ≤ 6 500 ₽, ASP ≥ 1 500 ₽, % выручки через пакеты | безопасность: 0 серьёзных происшествий, журнал инцидентов |
| Финал | 3,82 млн ₽ капитала, кнопка «Финмодель» | только продукт и роадмап |

Также убраны налоговый режим и формулировка stop-loss про возврат депозита и продажу инвентаря.

**Осознанно оставлено:** цены продукта (деловое знакомство 700 ₽, день 1 700 ₽, «Ритм 4/8/12») — команда должна знать оффер, который продаёт; и размер рынка (590 млрд ₽) как контекст категории, а не показатель проекта.

## Смотреть локально

```bash
python3 -m http.server 8099
# затем http://127.0.0.1:8099/delovoy-pes-team/
```

25 слайдов. Навигация: ← → / Space / PgUp / PgDn, клик по миниатюрам, `Home`/`End`, `R` — сброс. На телефоне — вертикальная лента.
