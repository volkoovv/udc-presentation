# Деловой пёс — презентация для команды

Версия общей презентации для найма, онбординга и работы с командой: продукт, день собаки, безопасность и стандарт работы — без коммерческих и финансовых блоков.

**Живая версия:** https://volkoovv.github.io/udc-presentation/delovoy-pes-team/

25 слайдов. Навигация: ← → / Space / PgUp / PgDn, клик по миниатюрам, `Home`/`End`, `R` — сброс. На телефоне — вертикальная лента.

## Что внутри

Зачем существует клуб, как устроен день собаки, безопасный старт и скрининг, роль кинологии, зонирование и благополучие, цифровой слой и отчёт владельцу, этапы запуска и критерии решения.

## Смотреть локально

```bash
python3 -m http.server 8099
# затем http://127.0.0.1:8099/delovoy-pes-team/
```

Открыть `index.html` напрямую (`file://`) не получится — деке нужен HTTP-сервер.

| Файл | Роль |
