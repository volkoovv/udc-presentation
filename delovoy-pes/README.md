# Деловой пёс — общая презентация

Интерактивная презентация проекта **«Деловой пёс»** — городского клуба для собак (Екатеринбург, пилот 2026).
Новая версия деки: ребренд, дизайн-система **Carbon Wasabi**, обновлённые фото и деловой тон.

**Живая версия:** https://volkoovv.github.io/udc-presentation/delovoy-pes/

Предыдущая версия (Urban Dog Club) осталась нетронутой: https://volkoovv.github.io/udc-presentation/

## Что изменилось относительно старой деки

| | Было | Стало |
|---|---|---|
| Бренд | Urban Dog Club | **Деловой пёс** · городской клуб для собак |
| Знак | CSS-квадрат с точкой | утверждённый вектор — пёс в тёмных очках (`assets/mark-bone.svg`) |
| Палитра | Urban Calm (петроль / оат / хурма / шалфей) | **Carbon Wasabi**: Carbon `#151918`, Mineral Bone `#F3F1E9`, Ink `#0E1211`, Wasabi `#C6EA4A`, Cool Blue `#B9D7E8`, Signal Coral `#FF5F3C` |
| Шрифты | Onest + Golos Text + JetBrains Mono | **Onest + Golos Text** (mono убран — его нет в системе) |
| Фото | рендеры Urban Calm | реальные рендеры Carbon Wasabi (интерьеры, съёмка процесса) |
| Слоган | — | **У каждой собаки свои дела** · «Пока вы на работе, пёс тоже при деле» |
| Лексикон | день знакомства | **деловое знакомство** |

Роли цвета: Wasabi — движение, маршрут, активные данные; Cool Blue — отдых; Signal Coral — единственный сигнал/CTA (<1%).

**Суть и структура не менялись** — те же 27 слайдов и та же логика: день как единица продукта, пилот в Екатеринбурге, юнит-экономика.

## Смотреть локально

```bash
python3 -m http.server 8099
# затем http://127.0.0.1:8099/delovoy-pes/
```

Открыть `index.html` напрямую (`file://`) не получится — деке нужен HTTP-сервер.

## Навигация

- **Десктоп:** ← → / Space / PgUp / PgDn, клик по миниатюрам, `Home`/`End`, `R` — сброс.
- **Планшет:** тап по правой/левой половине + лента миниатюр.
- **Телефон:** вертикальный скролл, все 27 слайдов лентой.

## Устройство

| Файл | Роль |
